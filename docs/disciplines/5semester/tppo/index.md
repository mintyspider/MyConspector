# Технология производства программного обеспечения

Лектор - Дмитрий Жоржевич Корзун. Дисциплина проводилась у студентов 3 курса направлений ПМиИ, ИСиТ и ПИ в 2025 году.

Здесь размещены общие ответы на вопросы к зачету.

- [drawio](https://www.drawio.com/) для построения схем

- [Соммервилл, И. Инженерия программного обеспечения. 6-е издание : Пер. с англ. / И. Соммервилл. – Москва: Издательский дом «Вильямс», 2002. 624 с. ISBN 5-8459-0330-0.](https://djvu.online/file/XSYvi7ZHnZ3RH)
