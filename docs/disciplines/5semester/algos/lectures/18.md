# 18. Полиномиальная и экспоненциальные сложности. Классы P и NP. Полиномиальная сводимость. NP-полные задачи. Примеры.

## Полиномиальная сложность (класс P)
::: info <span class="defn">Опр.</span> **Класс P** (от англ. polynomial) — множество задач, для которых существуют «быстрые» алгоритмы решения (время работы которых полиномиально зависит от размера входных данных). 
:::

Примеры задач:
- сортировка списка;
- поиск кратчайшего пути в графе (алгоритм Дейкстры);
- умножение матриц.
 
::: warning Важно
Любая задача, принадлежащая классу P, принадлежит и классу NP, так как может быть полиномиально проверена. Однако вопрос о равенстве классов P и NP остаётся нерешённым — научное сообщество склоняется в сторону отрицательного ответа.
:::

## Экспоненциальная сложность (класс NP)
::: info <span class="defn">Опр.</span> **Класс NP** (от англ. Nondeterministic Polynomial Time) — задачи, для которых проверка правильности решения может быть выполнена за полиномиальное время, но само решение может быть найдено только за экспоненциальное время (или хуже). 
:::

Примеры задач:
- задача о рюкзаке;
- задача коммивояжёра;
- задача выполнимости булевых формул (SAT).
 
::: warning Важно
Класс NP шире — он включает задачи, для которых можно быстро проверить правильность предложенного решения, но не обязательно быстро найти само решение. 
:::

## Полиномиальная сводимость
**Задача Z1 сводится к задаче Z2**, если метод решения задачи Z2 можно преобразовать в метод решения задачи Z1. 

::: info <span class="defn">Опр.</span> Сводимость называется **полиномиальной**, если указанное преобразование можно осуществить за полиномиальное время. 
:::

Пример: задача выяснения существования в ориентированном графе гамильтонового цикла, длина которого меньше или равна M, — NP-задача, так как рассмотрение каждого варианта требует полиномиального времени.

## NP-полные задачи
::: info <span class="defn">Опр.</span> **NP-полные задачи** (NP-complete) — задачи, которые одновременно являются NP-трудными и принадлежат классу NP. Это самые сложные задачи в классе NP. 
:::

Примеры:
- задача коммивояжёра;
- задача о раскраске графа;
- задача о выполнимости булевых формул (SAT).
 
Свойство: если для одной NP-полной задачи будет найден полиномиальный алгоритм, то все задачи в классе NP также смогут быть решены за полиномиальное время. 

::: warning Важно
Хотя решение NP-полной задачи может быть проверено «быстро», известного способа быстро найти решение не существует — время, необходимое для решения задачи с использованием любого известного алгоритма, быстро увеличивается по мере роста размера задачи. 
:::
