# 7. Определение жизненного цикла ИС

::: warning :small_orange_diamond: Кратко
Непрерывный процесс от принятия решения о создании ПО до его изъятия из эксплуатации
:::

<span class="defn">Опр.</span>  <highlight>**Жизненным циклом программного обеспечения**</highlight> называется непрерывный процесс, который начинается с момента принятия решения о необходимости создания программного обеспечения и заканчивается в момент его полного изъятия из эксплуатации.
