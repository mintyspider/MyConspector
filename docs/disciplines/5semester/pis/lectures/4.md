# 4. Классификация информационных систем по структурированности задачи

::: warning :small_orange_diamond: Кратко

+ **Структурированная задача**: известна вся информация, возможна полная автоматизация.
+ **Неструктурированная задача**: невозможно выделить элементы и связи, автоматизация ограничена.
+ **Частично структурированная задача**: известна часть информации, требуется участие человека.

:::

## <highlight>Структурированная (формализуемая) задача</highlight>

> задача, где известны все ее элементы и взаимосвязи между ними. В такой задаче удается выразить ее содержание в форме **математической модели**, имеющей точный алгоритм решения. Целью использования информационной системы для решения структурированных задач является **полная автоматизация** их решения.

## <highlight>Неструктурированная (неформализуемая) задача</highlight>

> задача, в которой невозможно выделить элементы и установить между ними связи. Решение таких задач из-за **невозможности создания математического описания и разработки алгоритма** связано с большими трудностями. Возможности использования здесь информационной системы **невелики**.

## <highlight>Частично структурированная задача</highlight>

> задача, в которой известна лишь часть элементов и связей между ними. Получаемая с помощью информационной системы **информация анализируется человеком**, который будет играть определяющую роль. Такие информационные системы являются автоматизированными, так как в их функционировании принимает участие человек.
