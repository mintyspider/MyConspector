# 12. Алгоритмы поиска подстроки. Простейший поиск.

## Алгоритмы поиска подстроки.

<span style="border: 2px solid black; padding: 3px; display: inline-block;">Опр.</span> Алфавит – некоторое множество (набор) символов.

<span style="border: 2px solid black; padding: 3px; display: inline-block;">Опр.</span> Строка – последовательность, состоящая из символов алфавита.

<span style="border: 2px solid black; padding: 3px; display: inline-block;">Опр.</span> Подстрока – строка, состоящая из **смежных** символов исходной строки.

Постановка задачи: нужно найти подстроку в строке.

К алгоритмам поиска подстроки относят, помимо простейшего поиска, *алгоритм Кнута-Морриса-Пратта* и использование *z-функции*.
## Простейший поиск

Пусть исходная строка $heystack$ содержит $n$ символов, а искомая $needle$ – $m$ символов.

Самый наивный способ нахождения подстроки: перебрать каждый символ $heystack$ до индекса $n - m$ и прочитать следующие $m$ символов. Если все $m$ символов совпали, то мы нашли $needle$.

```cpp
	for (i = 0; i < n - m + 1; i++){
		// Внутренний цикл проверяет, что следующие m символов совпадают 
		// с подстрокой
		bool areEqual = True; 
		for (j = 0; j < m; j++){
			if (haystack[i + j] != needle[j]){
				areEqual = False;
				break;
			}
		}
		// Цикл завершился, не найдя несовпадений:
		if (areEqual){
			cout << "Найдено начало подстроки на индексе: " << i << "\n";
		}	
	}		
```

## Асимптотика

Простейший поиск в худшем случае делает $m$ сравнений для $n - m + 1$ символа исходной строки. Т.е., его асимптотика: $O(mn)$.
