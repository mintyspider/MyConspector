# 23. Основные элементы диаграммы потоков данных DFD

- Процесс – функция по изменению данных
- Внешняя сущность – внешние системы, из которых поступает или куда направляется информация
- Хранилище данных – файл, репозиторий, таблица базы данных.
- Поток данных – направление и сами данные, которые перемещаются между процессами и хранилищами данных или между процессами и внешними сущностями или между процессами
