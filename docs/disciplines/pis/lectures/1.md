# 1. Определение информационной системы

::: warning :small_orange_diamond: Кратко
**Информационная система** (ИС) автоматизирует бизнес-процессы, обеспечивая их информацией.
:::

<span class="defn">Опр.</span> <u>**Информационной системой**</u> называется автоматизированная система, предназначенная для обеспечения бизнес-процессов информацией.

><u>Бизнес-процесс</u> - совокупность взаимосвязанных действий или задач, которые выполняются для достижения определённой цели компании.
