# 39. Каких правил следует придерживаться при оформлении интерфейса?

## Общие правлиа

- интерфейс - не самоцель;
- интерфейс передает информацию своему пользователю;
- интерфейс обычно предназначен для длительного использования;
- интерфейс функционален.

---

## Эстетика

- избегайте ярких цветов;
- добивайтесь контраста не сменой насыщенности элементов, а расположением пустот;
- в интерфейсах обычно не место красоте;
- используйте визуальные закономерности;
- используйте модульные сетки;
- привязывайте все размеры и координаты к золотому сечению (0.618 х 0.382).
