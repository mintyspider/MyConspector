# 2. Цели создания информационной системы

::: warning :small_orange_diamond: Кратко
**Цели создания ИС**: повышение эффективности, уменьшение затрат, рисков, интероперабельности, безопасности и управляемости.
:::

## Цели создания ИС

* Повышение эффективности бизнес-процессов (_основа_)
* Уменьшение затрат бизнес-процессов
* Уменьшение рисков бизнес-процессов
* Повышение интероперабельности( возможности и прозрачности взаимодействия) бизнес-процессов
* Повышение безопасности бизнес-процессов
* Повышение управляемости бизнес-процессами
