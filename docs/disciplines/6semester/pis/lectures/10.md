# 10. Что такое состояние?

**Набор конкретных значений атрибутов объекта.**

Состояние может быть задано в виде набора конкретных значений атрибутов класса или объекта, при этом изменение их отдельных значений будет
отражать изменение состояния моделируемого класса или объекта. Не каждый атрибут класса может характеризовать его состояние. Как правило, имеют значение только такие свойства элементов системы, которые отражают <highlight>динамический или функциональный</highlight> аспект ее поведения.
