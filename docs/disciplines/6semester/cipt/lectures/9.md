# 9. Блокчейн.

## Понятие блокчейна

Блокчейн — распределённая база данных, в которой информация хранится в виде последовательности связанных блоков.

Каждый блок содержит:
- данные;
- хеш текущего блока;
- хеш предыдущего блока;
- временную метку.

Связь блоков с помощью хешей обеспечивает целостность цепочки.

## Основные свойства блокчейна

1. Децентрализация

Данные хранятся одновременно на множестве узлов сети.

Отсутствует единый центр управления.

2. Неизменяемость данных

Изменение информации в одном блоке приводит к изменению его хеша и нарушению всей цепочки.

Это делает подделку данных крайне сложной.

3. Прозрачность

Все участники сети могут проверять состояние цепочки блоков и историю транзакций.

4. Криптографическая защита

Для защиты информации используются:
- хеш-функции;
- цифровые подписи;
- асимметричное шифрование.

## Структура блока

Типичный блок содержит:

1. Заголовок блока
- хеш предыдущего блока;
- хеш текущего блока;
- временную метку;
- nonce (специальное число для майнинга).

2. Тело блока
- список транзакций;
- служебную информацию.

## Типы блокчейнов

1. Публичный блокчейн
Доступен всем пользователям.

Пример: Bitcoin

2. Приватный блокчейн
Управляется одной организацией.

Доступ ограничен.

3. Консорциумный блокчейн
Управляется группой организаций.

## Преимущества блокчейна

- децентрализация;
- устойчивость к подделке данных;
- прозрачность;
- высокая отказоустойчивость;
- снижение роли посредников.

## Недостатки блокчейна

- низкая производительность;
- большие энергозатраты (PoW);
- сложность масштабирования;
- необратимость ошибок;
- большой объём хранения данных.

## Применение блокчейна

Блокчейн используется в:
- криптовалютах;
- банковских системах;
- логистике;
- электронном голосовании;
- хранении цифровых активов;
- системах идентификации.
