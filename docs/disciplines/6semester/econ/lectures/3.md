# 3. Эконометрическое исследование

## Методы эконометрики
- регрессионный анализ
- анализ временных рядов
- системы взаимозависимых (совместных, одновременных) уравнений
- статистические методы классификации и снижения размерности

## Этапы эконометрического исследования
1. Постановка проблемы
2. Спецификация модели
3. Получение статистических данных
4. Оценка параметров модели
5. Верификация модели
6. Интерпретация результатов
