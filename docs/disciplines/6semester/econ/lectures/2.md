# 2. Типы переменных в эконометрической модели

$$F(x,y,a,\epsilon)=0$$

- x - экзогенные переменные (те, по которым предсказываем)
- y - эндогенные переменные (те, которые предсказываем)
- $\epsilon$ - латентная переменная (случайная) - отражает влияние неучтенных факторов, а также ошибки измерений.

## Примеры моделей

$$y = a_0 + a_1 \cdot x + \epsilon$$

$$y = a_0 + a_1 \cdot e^x + \epsilon$$

$$y = \dfrac{a_1\cdot x_1}{a_2\cdot x_2 + a_3} + \epsilon$$

и т.д.

## Типы переменных
- Количественные
- Порядковые (ординальные) – оценка (неудовлетворительно, удовлетворительно, хорошо, отлично), сорт товара (II, I, высший)
- Категориальные (классификационные, номинальные, качественные) – пол (женский, мужской), цвет (синий, красный, зеленый, белый), семейное положение (холост, женат, разведен, вдовец)
